#### 6.2 汇丰 HSBC One（需亲身过港）

- **预约：** HSBC HK 官网预约开户
- **材料：** 身份证 + 通行证 + 过关小票 + 地址证明 + **建议带工资流水/税单/投资经验证明**
- **注：** 2025-2026 年香港银行开户审核趋严，柜员说"投资"可能不够。建议准备充分的资金来源证明

- **美元账户用途：** 收到港币后可在汇丰网银内换汇成美元，再以美元电汇至盈透，减少盈透内的换汇步骤和汇率损失

#### 资金流转路径（含额度提示）

```
内地银行卡（购汇，每人每年等值5万美元便利化额度）
    ↓
汇丰 HK（中转，建议同时开通港币+美元账户）
    ↓
盈透 IBKR（投资）
    ↓
ZA Bank（日常消费/备用）
```

**⚠️ 各环节转账限制：**
- 内地→香港：每人每年便利化额度等值 **5 万美元**，单笔等值 **1 万美元** 以上通常会被银行要求补充资金用途证明
- 香港→盈透（SWIFT 电汇/FPS）：一般无单笔额度限制，但大额转账可能触发盈透风控审核
- 香港→内地：大额资金回流需留意外汇申报要求
- ⚠️ **携带外币现钞出境规则：** ≤$5,000 无需手续；$5,000-$10,000 需向银行申领《携带外汇出境许可证》；> **$10,000 原则上禁止**。不建议通过携带大量现钞方式转移资金

#### ⚠️ 银行账户风控

- 所有资料必须真实
- 开户后不要快进快出，不接收陌生第三方转账
- 保持账户每月至少登录一次

**验收标准：** ZA Bank 开户成功，汇丰视情况而定（最低目标：ZA Bank+盈透链路跑通）
**降级方案：** 无法赴港 → 先用 ZA Bank 主力，汇丰后续补。**如果 ZA Bank 开户失败 → 转向工银亚洲（远程见证开户门槛较低）或永隆银行（招行旗下，内地客户经验丰富）。**

### #7 盈透证券开户（含合规说明）

#### ⚠️ 合规前置（必读）

1. **外汇管理：** 中国个人每年有 5 万美元便利化购汇额度，但购汇**申报用途不得填写「境外证券投资」**，直接电汇至境外券商存在政策合规风险。**实操常见路径：** 内地银行卡年度便利化额度内购汇（用途如实填写「因私旅游」「探亲」等合规选项）→ 跨境电汇至本人香港银行账户 → 从香港银行入金盈透。**注意：购汇用途与后续资金实际去向需自行评估合规风险，本文不构成合规建议。**
2. **CRS/FATCA：** 盈透开户需填 W-8BEN 表格声明非美国税务居民。填写后税收规则如下：
   - **股息预扣税：** 个人投资者适用中美税收协定优惠税率 **10%**（若不填 W-8BEN 则默认为 30%）。建议优先配置低分红 ETF（如 QQQM）减少股息税损
   - **资本利得税：** 非美国税务居民**免征**美国资本利得税（买卖股票差价不交税）
   - **中国税务端：** 中国税务居民需就全球收入（含股息和资本利得）申报个人所得税，境外已缴税款可申请税收抵免
   - 中国已实施 CRS，境外金融账户余额及收益信息会被交换回国内税务机关
   - **建议保留完整的交易记录和资金来源证明**

#### 注册步骤

1. ibkr.com → 注册。**账户主体选择（重要）：**
   - **IBKR LLC（美国主体）**：交易品种最全，入金仅支持 SWIFT 电汇
   - **IBKR HK（香港主体）**：支持 FPS（转数快）本地入金，费用低到账快，交易美股/ETF 功能无差异
   - **新手建议选 IBKR HK，走 FPS 入金更顺畅**
2. 填写信息与护照一致，投资经验如实填写
3. **账户类型：** 先开 **Cash（现金账户）**，不要开 Margin（保证金有杠杆）
4. **交易权限：** 先只勾 Stocks，Options（期权）暂不开通
5. 首笔入金建议 ≥ $1,000-2,000

**⚠️ 投资经验与权限的关系：**
- 如果选择投资经验为"None"，部分交易权限（如期权、保证金）可能被系统自动拒绝
- 这属于正常现象，后续补救方法：
  1. 入金后先做 ETF 交易积累实盘记录
  2. 1-2 个月后在盈透后台重新提交交易经验申请
  3. 或联系盈透客服提交模拟盘交易记录作为经验证明

#### 入金方式

**路径一：香港银行 → 盈透（推荐）**
- **按账户主体选择入金方式：**
  - 选择 **IBKR HK（香港主体）** → FPS（转数快）本地转账，即时到账，手续费低
  - 选择 **IBKR LLC（美国主体）** → SWIFT 电汇至盈透美国收款行，1-3 个工作日，手续费 HKD $50-200/笔
- 注意：ACH（自动清算系统）是美国本土清算网络，**香港银行无法直接 ACH 至美国盈透**

**路径二：香港银行 → Wise → 盈透（仅限有海外居留权的用户）**
- 中国大陆身份注册的 Wise **无法获取美元 Routing Number**，此路径不可用
- 持有海外居留权者可走此路径，汇损约 0.5%

**路径三：内地银行 → 盈透（不推荐）**
- 内地银行跨境电汇手续费约 ¥150-300/笔 + 中间行扣费 $10-30
- **⚠️ 内地银行可能因汇款用途涉及投资而拦截或退回，建议先致电客服确认。首笔建议试汇 ≤ $500 测试通道**

**⚠️ 各银行电汇费用概览（以 ¥10,000 等值外币为例）：**
| 银行 | 手续费约 | 备注 |
|------|---------|------|
| 中国银行 | ¥200-250 | 手机银行常有手续费 8 折 |
| 招商银行 | ¥150-200 | 金葵花客户免手续费 |

### #21 交易学习（★完全重构版）

> **原第8条「拿1万块亏完」重构为分层学习路径。**
>
> **红线（不可逾越）：** 绝不借钱 / 不加杠杆 / 不用生活必需资金 / 禁止跳过模拟盘直接实盘

#### 资金分层管理（三独立心理账户）

| 账户 | 来源 | 用途 | 纪律 |
|------|------|------|------|
| **储蓄账户** | 月收入 30% | 应急备用金 | 雷打不动，货币基金/短债 |
| **投资账户** | 储蓄达标后剩余 | 长线定投 ETF | 只进不出，3-5 年周期 |
| **学习账户** | 额外拨出 ≤ ¥10,000 | 交易学习专属 | 亏完暂停，复盘后再补 |

#### 学习路径

**第1-2月（观察期）—— 模拟盘**
- 盈透 Paper Trading（模拟盘），标的限定 SPY / QQQ / VT
- 学习：市价单/限价单/止损单、K 线、技术指标基础
- 阅读《漫步华尔街》或《共同基金常识》建立认知
- **不下实盘单**

**第3-4月（ETF 实盘期）—— 小仓位**
- 投资账户资金实盘，只买宽基 ETF（VOO / VTI / QQQM）
- 练习定投策略，感受市场波动和心理承受力
- 每笔交易写日志：买入理由 + 持有逻辑 + 情绪状态

**第5-6月（进阶学习期）—— 模拟期权**
- 学习期权基础概念：Call/Put、行权价、到期日、Greeks
- 学习资源：Investopedia 期权入门 / Tastytrade 的期权课程 / YouTube "Options Trading for Beginners"
- 盈透期权交易界面较复杂，模拟盘先熟悉操作（选合约、输入数量、确认订单），避免实盘时操作失误
- **学习账户资金量（¥10,000 ≈ $1,400）只适合做期权买方（Long Call / Long Put）**，无法安全执行 Covered Call 或 Cash-Secured Put 等卖方策略（保证金要求高、风险大。卖方策略需在投资账户正股持仓充足时再考虑）
- **规则：** 实盘期权前必须有 ≥ 3 个月模拟盘盈利记录

**⏰ 美股交易时间提示：**
- 夏令时（3 月中-11 月初）：北京时间 **21:30-04:00**
- 冬令时（11 月初-3 月中）：北京时间 **22:30-05:00**
- **不要熬夜盯盘，** 善用盈透的条件单（Conditional Orders）和止损单（Stop Loss）让系统自动执行

#### ⚠️ 风控规则

- **最大回撤控制：** 学习账户总回撤达到 **10%** 时暂停交易，复盘后再继续。默认单笔亏损不超过总资金 **2%**
- **硬性止损（与回撤联动）：** 每月亏损达到学习账户总资金 **20%** 时，当月禁止交易（这已经是底线，建议以 10% 为行动线）
- 每笔交易下单前设定止损价，**不设止损不下单**
- **⚠️ PDT 规则（Pattern Day Trader）：** 账户资产 < **$25,000** 时，5 个交易日内日内交易（同一标的当天买+卖）不得超过 3 次，否则账户被冻结 90 天。**学习账户仅 $1,400，极容易触发。建议只做波段（持仓过夜），或在盈透后台开启「PDT 保护」功能**
- **修改目标认知：** 交易的目的不是「亏完交学费」，而是「建立交易系统」。1 万块亏完和 1 万块亏损控制在 10% 以内，后者学到的多 10 倍。

#### 交易日志模板

```
日期：YYYY-MM-DD
标的：SPY
方向：买入 Call / 买入正股
仓位占比：__%（占总资金比例，单笔建议 ≤ 2%）
入场理由：______（技术面/基本面/情绪面）
入场价：$____
止损价：$____（预设）
出场价：$____
盈亏：+/- $____
持有时间：__天
心理状态评分（1-10）：__
错误总结（如有）：______
```

**验收标准：** 交易日志满 20 笔，能清晰说出自己的交易策略和常见错误

### #29 跨境收入税务合规（★新增）

> 涉及跨境收入（Upwork 接单、产品付费、投资收益）时必须了解：

**基本原则：**
- 中国税务居民需就**全球收入**申报个人所得税
- 境外收入已缴税款可申请税收抵免（避免双重征税）
- 外汇收入受个人年度便利化额度管理（5 万美元）

**建议：**
- 使用合规跨境收款通道（Payoneer / Wise），保留完整交易记录
- ⚠️ **Payoneer/Wise 结汇风险：** 频繁大额结汇到内地银行卡可能触发银行反洗钱风控（AML），导致银行卡被冻结。如果收款金额较大（月入 > $1,000），建议：
  - 若为货物贸易类跨境收入，可注册个体工商户开立外汇结算账户结汇，不占用个人额度。**注意：适用于货物贸易，不适用于 Upwork 等个人劳务/服务收入；银行审核要求真实贸易材料（报关单、平台后台数据等），并非仅凭订单截图**
  - 或使用 XTransfer / PingPong 等 B2B 外贸收款工具
- 年度收入超过一定额度时咨询专业税务顾问
- 中港税收协定：香港收入在香港缴税后可申请内地税收抵免

### #30 失败预案与退出机制（★新增）

> 不假设计划完美执行，提前想好调整策略。

**进度落后应对：**
- 单月完成率 < 70% → 压缩非核心条目，只保「英语 + 输出 + 储蓄」三项底线
- 连续 2 个月完成率 < 50% → 暂停计划，花 1 周重新评估
- 香港银行开户被拒 → 先用 ZA Bank，汇丰择期再试；盈透入金改用 Wise 中转
- X 运营 3 个月净增长 < 50 关注 → 60 天试错期，无正反馈则换赛道或平台

**三类重要应急场景：**

| 场景类型 | 具体风险 | 预案 |
|---------|---------|------|
| **账号丢失** | Apple ID 被锁 / X 被封 / 银行冻结 | 开启双重验证 + 救援邮箱 + 手机号绑定；关键凭证（身份证/护照/银行卡号）离线纸质备份 |
| **设备损坏** | 工作电脑/手机突然无法使用 | 代码和笔记及时推送或同步云端（Obsidian 同步 iCloud / Git 仓库）；备用设备方案准备 |
| **政策变化** | 购汇政策收紧 / 境外券商监管变化 | 关注国家外汇管理局官网动态；预留 3-6 个月的资金缓冲；备选方案（港股通、QDII 基金）提前了解 |

**退出条件：**
- 任何一条合规红线被触发（如违规境外投资导致账户冻结）
- 交易学习账户亏空且出现借贷行为
- 连续 3 周出现严重倦怠（不是懒，而是无法恢复的疲惫）

**退出后：** 复盘原因 → 调整计划 → 重新启动。180 天做不完，200 天做完也是胜利。

**心理预期管理：**
- 执行过程中挫折和停滞是常态，不是个人失败
- 任何条目执行困难时都可以 **降级执行**（如每天输出 300 字做不到就先写 100 字，100 字也做不到就写 50 字）
- 降级 ≠ 放弃。保住了连续性，就是保住了长期成功的最大变量

**习惯叠加（Habit Stacking）—— 降低启动阻力：**
将新习惯与已有习惯绑定，不需要额外意志力：

| 已有习惯/场景 | 叠加的新习惯 |
|-------------|-------------|
| 早晨喝咖啡/茶 | 同时刷 15 分钟 HackerNews / RSS（阅读） |
| 通勤 / 坐地铁 | 用 Tandem 聊 10 分钟英语 或 听英文 Podcast（听力） |
| 晚上洗澡前 | 跳绳 5 分钟 / 50 个深蹲（健身） |
| 午饭后 | 随手记 Flomo（输出） |
| 睡前 | 复盘当日消费 → 钱迹记账（记账） |

**精力管理（而非单纯时间管理）：**
将任务按认知负荷匹配到对应精力时段：

| 时段 | 精力状态 | 适合任务 |
|------|---------|---------|
| 早晨（高峰期） | 高 | 写代码 / 深度阅读 / 写长文 |
| 下午（平台期） | 中 | AI 对话 / 英语听力 / 产品迭代 |
| 晚间（低谷期） | 低 | 记账 / 刷 RSS / 整理 Obsidian 收件箱 / 运动 |

**上班族适配版：**
```
通勤（30min，低负荷）：听英语 Podcast / 刷 RSS
午休（20min，中负荷）：读技术文章 + 随手记 Flomo
晚 8-10pm（高负荷）：写代码 / 写长文 / 交易复盘
```
